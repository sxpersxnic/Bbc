Authentication & Security

  • User Story 1:

    As a user, I want my password to be encrypted for stronger security.

  • User Story 2:

    As a user, I want my personal data to be securely stored in a database.

  • User Story 3:

    As a user, I want meaningful error messages for login or registration failures (e.g., empty login field, incorrect password, invalid email during login; existing email during registration; weak password length) to improve the user experience.

  • User Story 4:

    As a user, I want a password confirmation field during registration.

  • User Story 5:

    As a user, I want meaningful error messages for login or registration failures (e.g., empty login field, incorrect password, invalid email during login; existing email during registration; weak password length) to improve the user experience.

  • User Story 6:

    As a user, I want a password confirmation field during registration.

  • User Story 7:

    As a user, I want to register and provide a username to create a new account.

  • User Story 8:

    As a user, I want to receive a message when the login is successful.

  • User Story 9:

    As a user, I want to log in using my email OR username.

  • User Story 10:

    As a user, I want to log in using my email.

  • User Story 11:

    As a user, I want to receive a message when the login is successful.

  • User Story 12:

    As a user, I want to log in to access the services.

  • Optional User Story 13:

    As user I would like to reset my password if I have forgotten my password

  • Optional User Story 14:

    On failed validation: Pay attention to domain endings when registering, special characters for password.

Messaging

  • User Story 15:

    As a user, I want a list to see all my existing chats.

  • User Story 16:

    As a user, I want to be able to create new chats with other users.

  • User Story 17:

    As a user, I want to ensure that chats with other people are not read by third parties (End-to-End Encryption).

  • User Story 18:

    As a user, I want to easily find a text field to compose messages.

  • User Story 19:

    As a user, I want to send a pre-typed message using both the send button and the Enter key.

  • User Story 20:

    As a user, I want messages to be delivered instantly, in real-time.

  • User Story 21:

    As a user, I want to retain old messages in the chat.

  • User Story 22:

    As a user, I want to be able to delete my messages.

  • User Story 23:

    As a user, I want to see with whom I am chatting.

  • User Story 24:

    As a user (Person A), I want to create a chat with Person B only once.

  • User Story 25:

    As a user, I don't want the other person to delete my messages (different rules apply in groups).

  • OptionalUser Story 26:

    As a user, I want to know if my message has been delivered and read.

  • OptionalUser Story 27:

    As a user, I want to edit messages that have already been sent.

  • OptionalUser Story 28:

    As a user, I don't want others to edit my messages.

  • OptionalUser Story 29:

    As a user, I want to see the names of message authors in group chats.

  • OptionalUser Story 30:

    As a user, I want to create group chats with multiple people.

  • OptionalUser Story 31:

    As a user, I want to be identified as the owner in the group chats I create.

  • OptionalUser Story 32:

    As a user, I want to manage the group chats where I am the owner.

  • OptionalUser Story 33:

    As a user, I want the ability to remove or block chats/contacts.

  • OptionalUser Story 34:

    As a user, I want to pin important chats.

  • OptionalUser Story 35:

    As a user, I want to add self-created tags to different chats.

  • OptionalUser Story 36:

    As a user, I want to receive notifications for new messages.

Groups

  • User Story 37:

    As a user I would like to be able to upload files to Chanels if I have the permissions.

  • User Story 38:

    As a moderator, I want to be able to moderate messages from all users to keep intolerant topics out of the chat.

  • User Story 39:

    As a moderator, I want to pin messages in different channels.

  • User Story 40:

    As a user, I want to react to messages to indicate whether I agree or disagree without creating a separate message.

  • User Story 41:

    As a moderator, I want to be able to change channel settings after creation, rather than creating a new channel if I’m unhappy with the initial settings.

  • User Story 42:

    As a user, I want to be able to open uploaded files in the app and then close them without being reset to the beginning of the directory.

  • User Story 43:

    As an owner, I would like to be able to create my own roles with different rights to give people different rights if they do not need/have all the rights an admin has but still need some functions that normal users do not have.

  • User Story 44:

    As an owner, I would like to be able to create, edit and delete tasks in order to create, edit and delete tasks.

  • User Story 45:

    As a user I would like to see my tasks sorted by submission date> to see which task has higher priority.

  • User Story 46:

    As a user, I would like to be able to submit my tasks in order to submit my work to the task owner.

  • User Story 47:

    As a user, I would like to be notified when an admin returns my task so that I don't always have to check myself when my work is returned.

  • User Story 48:

    As a user, I would like to be able to attach files to my tasks so that I can submit all my work.

  • User Story 49:

    As a user, I would like to be able to remove attached files if I accidentally attach the wrong file, so as not to submit unnecessary or confusing items.

  • User Story 50:

    As a user, I would like to return returned assignments in an improved form so that I can give my improved work to the assignment provider.

  • User Story 51:

    As a user, I would like to be able to undo the submission shortly after submission to prevent me from accidentally submitting too early.

  • User Story 52:

    As user I would like to be able to create groups

  • User Story 53:

    As owner, i would like to assign roles in a group i have created in order to give different people different rights.

  • User Story 54:

    As an owner, I would like to be notified when a USER returns a task to me, so that I can correct it as quickly as possible.

  • User Story 55:

    As an owner, I would like to be able to undo my return shortly after submission> to prevent me from accidentally submitting my return too early.

  • User Story 56:

    As an owner, I would like to return submitted tasks to distribute points or suggest improvements

  • User Story 57:

    As a moderator, I want to create various channels with names (optional) and a brief description, and make them accessible to different users.

  • User Story 58:

    As a moderator, I want to independently set permissions for users to write in created channels and view them, especially for informative channels.

  • User Story 59:

    As a moderator, I’d like an automatic content filter that screens messages containing specific words, making my job easier.

  • Optional User Story 60:

    As a moderator, I would like to define who can upload files.

  • Optional User Story 61:

    As a moderator, I want to create different types of channels with various functions.

Usage and management

  • Optional User Story 62:

    As a user, I would like to be able to edit my user profile to update my personal information.

Assignments

  • User Story 63:

    As a user, I would like to be able to determine how a quiz is presented (multiple choice etc.)

  • User Story 64:

    As a user, I would like to be reminded 2 hours before the deadline so that I don't forget to hand in a completed assignment.

  • User Story 65:

    As a user, I would like to be able to create, update and delete a quiz

  • User Story 66:

    As a user, I would like to be able to view the evaluation of my test.

  • User Story 67:

    As a user, I would like to be able to participate in courses and view the learning sets to see whether it is the learning set I want to learn.
