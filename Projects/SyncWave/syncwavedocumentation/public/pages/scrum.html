1st Sprint

Sprint Description

With the first sprint, the project begun. We met in our groups and discussed what our project should be about and what we would like to achieve. We decided to build a Teams like clone, so a team management platform.

Goals

To take the very first step of our application, we had to set up the Git repositories and the development enviornments. When this was done, we started with our first feature: Authentication. This meant for the backend team, to implement the logic of Signup and Signin. When a Sign up request gets received, the application should validate the credentials provided and if the user is not already existing in the database. Then the user should be created and added. When a Sign in request gets received, the application checks, if the user exists and if the login credentials are valid. If so then a JWT token needs to be generated.
